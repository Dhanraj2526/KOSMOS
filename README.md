# 🌌 KOSMOS

**Decentralized Asset Ecosystem**

![Status](https://img.shields.io/badge/Version-3.0%20Elite-14f0ff?style=for-the-badge)

## 🚀 Features
- **Secure Sidepanel:** Browser-integrated decentralized uploader.
- **Internal Previewer:** High-end viewer for Images & PDFs.
- **Blockchain Integrity:** Immutable asset anchoring on Ethereum.

## 🛠️ Setup
1. `npm install`
2. `cd client && npm install`
3. `npx hardhat node`
4. `npm start`
